## Problem 2 — Reverse the Array

### Problem Summary
The problem requires reversing an array of N integers and printing them in reverse order.

### Algorithm Explanation
1. Read integer N.
2. Store elements in a vector.
3. Traverse the vector from the last index to the first.
4. Print elements during reverse traversal.

### Time Complexity
- Input: O(N)
- Reverse traversal: O(N)
- Overall: **O(N)**

### Space Complexity
- Vector storage: **O(N)**
